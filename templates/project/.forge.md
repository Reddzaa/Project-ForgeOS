# Forge metadata

Machine-readable project state. Skills read and update the frontmatter above;
you can edit it by hand at any time.

**Field reference**

- `status`: `idea` → `active` → (`paused` | `maintenance`) → `shipped` | `parked` | `killed`
- `scope`: `weekend` (default — a few sessions), `mvp` (a few weeks), `product` (long-lived)
- `stack`: short slug once chosen, e.g. `tauri-react-ts`, `python-script`
- `forge_repo`: where skills find the global `memory/` folder. If the path is
  wrong or missing, skills work without memory and say so.
