# Principles

The opinions that make Forge OS what it is. When a skill instruction and a
principle conflict, the principle wins — and the skill should be fixed.

1. **Finished beats perfect.** A shipped weekend tool beats an abandoned
   masterpiece. Every default — scope, plan size, phase count — is tuned for
   finishing.

2. **Humans decide, AI suggests.** Skills recommend, draft, and log. The
   human owns verdicts, decisions, and everything promoted into memory.

3. **Documents are the shared state.** Chat history is disposable; the
   `forge/` folder is not. Anything worth keeping goes in a file — an
   unlogged session never happened.

4. **Memory must be curated.** Raw history is cheap and stays in projects.
   Global lessons are few, evidence-backed, human-confirmed, and pruned on a
   budget. A memory that only grows is a memory nobody reads.

5. **Keep small projects small.** Weekend scope is the default, and the
   process tax on it is measured in minutes. Bureaucracy that small projects
   route around protects nothing.

6. **Decisions need reasons — and expiry.** Every significant decision is
   logged with its rejected options and a concrete reconsider-when trigger.
   A decision without a facit teaches nothing.

7. **Errors should become lessons.** An error that cost two hours and was
   captured is an investment; the same error uncaptured is a subscription.

8. **Prove the shipping path first.** Phase 0 ends with an installable,
   deployable, runnable artifact. Projects die at deploy — so deploy on
   day one.

9. **Killing a project is a valid outcome.** Definition of dead is written
   at the start; parked requires a resume condition; killed gets a retro.
   The only failure mode is fading out silently.

10. **Subtract by default.** No cloud, auth, sync, database, or AI unless
    the brief demands it. Every dependency is friction now and breakage
    later.

11. **Agents are replaceable; files are not.** Nothing in the system depends
    on a specific AI vendor. If it can read files and follow instructions,
    it can run Forge OS — including whatever replaces today's agents.
